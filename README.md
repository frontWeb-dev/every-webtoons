## 프로젝트 설치

1. 패키지 설치

```
npm install
```

2. 프로젝트 실행

```
yarn dev
```

---

## 기능 시연

- 회원 가입, 로그인, 로그아웃을 할 수 있습니다.
- 웹툰을 요일별, 서비스별로 모아 볼 수 있습니다.
- 웹툰의 제목이나 작가명으로 검색을 할 수 있습니다.
- 웹툰의 상세 정보를 볼 수 있고, 보러가기를 클릭하면 해당 웹툰 사이트로 이동합니다.
- 관심 웹툰을 저장할 수 있습니다.
- 프로필 사진을 변경할 수 있습니다.
